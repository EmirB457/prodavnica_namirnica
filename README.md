# Sistem za upravljanje zalihama u prodavnici namirnica

Web aplikacija razvijena u Pythonu pomoću Streamlit biblioteke. Aplikacija omogućava dodavanje proizvoda, pregled proizvoda, prikaz niskog lagera i statistički pregled zaliha.

## Tehnologije
- Python
- Streamlit
- Pandas
- Plotly
- SQLite

## Pokretanje
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python -m streamlit run app.py
